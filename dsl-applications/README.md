# Dify DSL 应用集合

此目录存放通过 AI 对话创建的各种 Dify DSL 应用。

每个应用一个独立目录，包含：
- `*.yml` - DSL 配置文件
- `README.md` - 使用说明文档

## 📚 已有应用

### 1. [长篇小说生成器](./novel-generator/)

**DSL 文件**：`novel_generator.yml`  
**类型**：advanced-chat  
**模型**：DeepSeek-Chat  

**功能**：完整的长篇小说创作流程，包括世界观设定、大纲规划、角色创建、章节生成（3000-5000字）和自动审查润色。

**适用场景**：奇幻、科幻、武侠、现代等各类小说创作

## 📝 使用方式

### 导入应用

在 Dify 控制台：
1. 创建应用 → 导入 DSL
2. 选择应用目录下的 `.yml` 文件
3. 点击导入

### 创建新应用

直接告诉 AI 您的需求，例如：
```
我想创建一个 XXX 应用，功能是...
```

AI 会自动生成：
- DSL 配置文件（.yml）
- 使用说明文档（README.md）
- 自动导入到 Dify
